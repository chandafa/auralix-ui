# New Unique Components Walkthrough

I have added 5 new components inspired by React Bits and Magic UI to the Aural UI library.

## Components Added

### 1. Animated Gradient Text

A shimmering text effect for highlighting special content.

- **Location**: `src/components/ui/AnimatedGradientText.tsx`
- **Preview**: [Animated Gradient Text Page](http://localhost:3000/components/animated-gradient-text)

### 2. Border Beam

A moving gradient beam that travels around the border of a container.

- **Location**: `src/components/ui/BorderBeam.tsx`
- **Preview**: [Border Beam Page](http://localhost:3000/components/border-beam)

### 3. Dock

A macOS-style interactive dock that magnifies icons when hovered.

- **Location**: `src/components/ui/Dock.tsx`
- **Preview**: [Dock Page](http://localhost:3000/components/dock)

### 4. Meteor Meteors

A background effect simulating falling meteors.

- **Location**: `src/components/ui/MeteorMeteors.tsx`
- **Preview**: [Meteor Meteors Page](http://localhost:3000/components/meteor-meteors)

### 5. Magic Card

A card that reveals a spotlight gradient on hover that follows the mouse cursor.

- **Location**: `src/components/ui/MagicCard.tsx`
- **Preview**: [Magic Card Page](http://localhost:3000/components/magic-card)
